# ebhelper_asg_default_instance_warmup

Configures the default instance warmup period on an EB-managed Auto Scaling Group. During warmup, a newly launched instance is not counted toward aggregated metrics for scaling decisions. On destroy, sets to -1 (disabled, uses default cooldown instead).

## Example Usage

```terraform
resource "ebhelper_asg_default_instance_warmup" "app" {
  asg_name       = ebhelper_environment_info.app.asg_name
  warmup_seconds = 120
}
```
