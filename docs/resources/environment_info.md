# ebhelper_environment_info

Discovers AWS Elastic Beanstalk environment infrastructure (ASG, load balancer, target groups) during `terraform apply`. Acts as a deferred data source that resolves after the EB environment is created.

This resource does not create any infrastructure — it only discovers and exposes attributes for use by downstream resources.

## Example Usage

```terraform
resource "ebhelper_environment_info" "app" {
  application_name = "my-app"
  environment_name = "my-app-production"
}

# Use the discovered ASG name with native resources
resource "aws_autoscaling_schedule" "scale_up" {
  autoscaling_group_name = ebhelper_environment_info.app.asg_name
  scheduled_action_name  = "scale-up"
  recurrence             = "0 7 * * MON-FRI"
  desired_capacity       = 2
  min_size               = 2
  max_size               = 4
  time_zone              = "Pacific/Auckland"
}
```
