# ebhelper_asg_instance_maintenance_policy

Configures the instance maintenance policy on an EB-managed Auto Scaling Group, controlling instance replacement behavior during updates. On destroy, removes the policy.

- `max_healthy_percentage = 100`: Terminate before launch (no extra capacity)
- `max_healthy_percentage > 100`: Launch before terminate (extra capacity during replacement)

## Example Usage

```terraform
resource "ebhelper_asg_instance_maintenance_policy" "app" {
  asg_name               = ebhelper_environment_info.app.asg_name
  min_healthy_percentage = 90
  max_healthy_percentage = 120
}
```
