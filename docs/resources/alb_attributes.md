# ebhelper_alb_attributes

Manages ALB attributes on an EB-created Application Load Balancer. This allows modifying ALB settings that are not exposed through EB option settings and cannot be managed via `aws_lb` (since EB owns the ALB).

Only tracks attributes specified in configuration — AWS default attributes are not managed.

## Example Usage

```terraform
resource "ebhelper_alb_attributes" "app" {
  load_balancer_arn = ebhelper_environment_info.app.load_balancer_arns[0]

  attributes = {
    "routing.http.xff_client_port.enabled"                       = "true"
    "dns_record.client_routing_policy"                           = "any_availability_zone"
    "routing.http.x_amzn_tls_version_and_cipher_suite.enabled"   = "true"
  }
}
```
