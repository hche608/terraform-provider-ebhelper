# ebhelper_asg_max_instance_lifetime

Configures the maximum instance lifetime on an EB-managed Auto Scaling Group. Instances are terminated and replaced after this duration, enabling automatic instance rotation. On destroy, sets to 0 (disabled).

## Example Usage

```terraform
resource "ebhelper_asg_max_instance_lifetime" "app" {
  asg_name                     = ebhelper_environment_info.app.asg_name
  max_instance_lifetime_seconds = 604800  # 7 days
}
```
